# SOLOMYCRM Landing

Landing page comercial para presentar los servicios de SOLOMYCRM.

## Requisitos

- Node.js 18+
- npm 9+

## Comandos

- Desarrollo: `npm run dev`
- Build de produccion: `npm run build`
- Preview local del build: `npm run preview`

## Estructura principal

- `src/main.js`: contenido y secciones de la landing
- `src/style.css`: estilos y animaciones
- `index.html`: metadatos y contenedor raiz

## Mensaje comercial actual

La pagina presenta dos paquetes:

- Plan Individual: una persona realiza prospeccion y cierre.
- Plan Equipo: dos prospectores alimentan pipeline y un closer cierra ventas.
